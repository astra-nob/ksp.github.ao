ADCA Course Curriculum

Introduction to Computers

  • Evolution of computers and their applications
  • Hardware: CPU, input/output devices, memory, and storage
  • Software: Application, system, utility software, open-source vs proprietary software, mobile apps

Operating Systems

  • Basics of operating systems
  • File and folder management
  • Desktop settings and using control panels

Word Processing

  • Creating, editing, and formatting documents
  • Using tables, images, charts, and graphics
  • Mail merge and document security

Spreadsheets

  • Creating and editing spreadsheets
  • Applying formulas and functions
  • Data analysis and creating charts

Presentations

  • Creating and editing presentations
  • Slide design and layouts
  • Using animations and transitions

Internet and Web Browsing

  • Basics of the internet
  • Using web browsers and managing emails
  • Introduction to social networking platforms

E-Governance Services

  • Accessing online government services

Digital Financial Services

  • Using digital payment methods like UPI, BHIM, mobile wallets, and net banking
  • Understanding cybersecurity measures in online transactions

Communication and Collaboration Tools

  • Using online tools for collaboration (e.g., video conferencing, shared documents)
